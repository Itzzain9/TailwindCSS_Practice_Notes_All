Steps to install tailwind via NPM

  1. Npm init -y
  2. then npm install -D tailwaind postcss autoprefixer vite
  3. npm tailwind.config.js
  4. then in tailwind file, in content:[] add * to this content so that it will applly tailwind to every file prenet in that folder
  5. then in package json ---> in script test; echo then reply by it by "start"; "vite"
  6. then create a input.css file and add @tailwind base; @tailwind components; @tailwind utilites;
  7. then type npx tailwind init -p to generate post css config file.
